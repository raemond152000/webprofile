Recipe App


React, Redux, Axios, Firebase, Git ,JIRA

This is a group project for our Application course that aims to produce a full functioning app from start to finish utilizing SCRUM agile framework. Our app is a Recipe App that allows users to filter through and add recipe according to their needs. My role for this project was to add the functionality where a user can bookmarked or saved the recipe to their favourite list, so that when logged in the list would be retrieved. I've used firestore database to create and authenticate users and add the favourite list field to the users collection. We've used git version control and github for git repository. We also used JIRA to keep track of our project using Vision boards, Stories and Sprints. Working with a team proves not only to be effecient but fun too. My groupmates were very responsive and helpful. Everyone was up to their task and made learning more fun.

Vortex


React Native, Matter JS, Sprite Sheet Android Studio, Expo

A "space" game, it's objective is to reach to the mother nucleus avoiding collision with space objects. I used react native library for this project and matter JS for the physics for this game such as object collision and detection. I've used spritesheet for the images and animations. It was challenging yet a fun experience for me to learn the library and get the physics right. Setting up and learning the android studio emulator and the Expo go proves to be challenging as well yet rewarding when I get to play the game on my phone. I've gained alot of respect to how the games are made no matter how simple looking it is.
